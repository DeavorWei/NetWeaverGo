# 项目样式架构分析与统一管理方案

## 一、当前架构分析

### 1. 现有技术栈

- **CSS 框架**: Tailwind CSS v4（使用新语法 `@theme`, `@custom-variant`）
- **构建工具**: Vite + PostCSS
- **UI 框架**: Vue 3 + TypeScript

### 2. 当前样式结构

```
frontend/src/
├── style.css          # 全局样式入口（Tailwind 配置 + CSS 变量）
├── App.vue            # 主题切换逻辑
└── views/
    └── Tools/
        ├── ConfigForge.vue   # 大量内联样式 + scoped style
        └── NetworkCalc.vue   # 内联样式 + scoped style
```

### 3. 发现的问题

| 问题               | 描述                                                               | 影响                           |
| ------------------ | ------------------------------------------------------------------ | ------------------------------ |
| **主题状态分散**   | 主题切换逻辑直接写在 App.vue，没有集中管理                         | 难以在其他组件中访问主题状态   |
| **颜色使用不一致** | 部分使用 CSS 变量（`bg-bg-dark`），部分硬编码（`bg-slate-800/40`） | 维护困难，主题切换效果不完整   |
| **缺乏扩展机制**   | 只支持 light/dark 两种模式，架构不支持新增主题                     | 无法添加自定义主题             |
| **组件样式耦合**   | 大量 `dark:` 前缀和硬编码颜色直接写在组件中                        | 组件复用性差，样式难以统一调整 |
| **无设计令牌系统** | 没有统一的语义化颜色、间距、圆角等设计令牌                         | 新增组件时样式选择困难         |

### 2. 主题配置架构

#### 2.1 CSS 变量分层设计

```css
/* 第一层：原始颜色值（Primitive Tokens） */
:root {
  --color-gray-50: #f8fafc;
  --color-gray-100: #f1f5f9;
  --color-blue-500: #3b82f6;
  /* ... */
}

/* 第二层：语义化令牌（Semantic Tokens）- 跟随主题变化 */
[data-theme="light"] {
  --color-bg-primary: var(--color-gray-50);
  --color-bg-secondary: var(--color-gray-100);
  --color-text-primary: var(--color-gray-900);
  --color-accent: var(--color-blue-500);
}

[data-theme="dark"] {
  --color-bg-primary: #0f1117;
  --color-bg-secondary: #161b27;
  --color-text-primary: #e2e8f0;
  --color-accent: #60a5fa;
}

/* 第三层：组件级变量（Component Tokens） */
.card {
  --card-bg: var(--color-bg-secondary);
  --card-border: var(--color-border);
  --card-shadow: var(--shadow-card);
}
```

### 3. 主题管理 Composable

```typescript
// composables/useTheme.ts
import { ref, computed, watch, onMounted } from "vue";

export type ThemeName = "light" | "dark" | string;
export type ThemeMode = ThemeName | "system";

const THEME_KEY = "netweaver-theme";
const DEFAULT_THEME: ThemeName = "dark";

// 全局状态（单例模式）
const currentTheme = ref<ThemeName>(DEFAULT_THEME);
const isInitialized = ref(false);

export function useTheme() {
  // 初始化主题
  const initTheme = () => {
    if (isInitialized.value) return;

    const savedTheme = localStorage.getItem(THEME_KEY) as ThemeName | null;
    if (savedTheme && isValidTheme(savedTheme)) {
      currentTheme.value = savedTheme;
    } else {
      // 检测系统偏好
      currentTheme.value = window.matchMedia("(prefers-color-scheme: dark)")
        .matches
        ? "dark"
        : "light";
    }

    applyTheme(currentTheme.value);
    isInitialized.value = true;
  };

  // 应用主题到 DOM
  const applyTheme = (theme: ThemeName) => {
    const html = document.documentElement;

    // 移除旧主题属性
    html.removeAttribute("data-theme");
    html.classList.remove("dark", "light");

    // 应用新主题
    html.setAttribute("data-theme", theme);
    if (theme === "dark") {
      html.classList.add("dark");
    }

    // 更新 meta theme-color
    const metaThemeColor = document.querySelector('meta[name="theme-color"]');
    if (metaThemeColor) {
      metaThemeColor.setAttribute(
        "content",
        theme === "dark" ? "#0f1117" : "#f8fafc",
      );
    }
  };

  // 切换主题
  const setTheme = (theme: ThemeName) => {
    currentTheme.value = theme;
    localStorage.setItem(THEME_KEY, theme);
    applyTheme(theme);
  };

  // 切换明暗
  const toggleTheme = () => {
    setTheme(currentTheme.value === "dark" ? "light" : "dark");
  };

  // 计算属性
  const isDark = computed(() => currentTheme.value === "dark");
  const themeName = computed(() => currentTheme.value);

  // 验证主题名称
  function isValidTheme(theme: string): boolean {
    return ["light", "dark"].includes(theme);
  }

  // 监听系统主题变化
  onMounted(() => {
    initTheme();

    const mediaQuery = window.matchMedia("(prefers-color-scheme: dark)");
    const handler = (e: MediaQueryListEvent) => {
      if (!localStorage.getItem(THEME_KEY)) {
        setTheme(e.matches ? "dark" : "light");
      }
    };
    mediaQuery.addEventListener("change", handler);
  });

  return {
    currentTheme,
    themeName,
    isDark,
    setTheme,
    toggleTheme,
    initTheme,
  };
}
```

### 4. 组件样式规范

#### 4.1 推荐的组件样式写法

**❌ 不推荐（硬编码颜色）：**

```vue
<div class="bg-slate-800/40 dark:bg-slate-700/60 text-slate-300">
```

**✅ 推荐（使用语义化变量）：**

```vue
<div class="bg-bg-secondary text-text-primary border border-border">
```

#### 4.2 通用组件样式类

```css
/* styles/components/card.css */
.card {
  background-color: var(--color-bg-secondary);
  border: 1px solid var(--color-border);
  border-radius: var(--radius-lg);
  box-shadow: var(--shadow-card);
}

.card-header {
  padding: var(--spacing-4) var(--spacing-5);
  border-bottom: 1px solid var(--color-border);

.card-body {
  padding: var(--spacing-5);
}

/* styles/components/button.css */
.btn {
```

## 三、新增主题扩展机制

### 1. 添加新主题步骤

**步骤 1：定义主题变量**

```css
/* styles/themes/high-contrast.css */
[data-theme="high-contrast"] {
  --color-bg-primary: #000000;
  --color-bg-secondary: #1a1a1a;
  --color-text-primary: #ffffff;
  --color-text-secondary: #e0e0e0;
  --color-accent: #ffff00;
  --color-border: #ffffff;
}
```

**步骤 2：注册主题**

```typescript
// composables/useTheme.ts
const AVAILABLE_THEMES = ["light", "dark", "high-contrast"] as const;

function isValidTheme(theme: string): boolean {
  return AVAILABLE_THEMES.includes(theme as any);
}
```

**步骤 3：添加主题选项**

```vue
<!-- ThemeSwitch.vue -->
<template>
  <select v-model="currentTheme" @change="setTheme($event.target.value)">
    <option value="light">明亮模式</option>
    <option value="dark">黑暗模式</option>
    <option value="high-contrast">高对比度</option>
  </select>
</template>
```

### 2. 主题配置文件格式（可选扩展）

```typescript
// config/themes.ts

export const themes: Record<string, Theme> = {

  },

    colors: {
      bg: { primary: "#0c4a6e" /* ... */ },
      accent: { primary: "#06b6d4" },
    },
  },
};
```

## 五、总结

| 方面     | 现状                 | 目标                       |
| -------- | -------------------- | -------------------------- |
| 主题管理 | 分散在 App.vue       | 集中在 useTheme Composable |
| 颜色定义 | 混合使用变量和硬编码 | 统一使用语义化 CSS 变量    |
| 扩展性   | 仅支持 light/dark    | 支持无限扩展新主题         |
| 组件样式 | 内联 Tailwind 类     | 语义化类 + 组件样式文件    |
| 维护性   | 难以统一修改         | 集中管理，易于维护         |
