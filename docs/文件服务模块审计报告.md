## 1. 模块概述

文件服务模块提供内置轻量级文件服务器功能，支持SFTP、FTP、TFTP、HTTP四种协议，主要用于网络设备配置文件传输、固件升级等场景。

### 5.3 潜在问题

#### ⚠️ 低风险：TFTP停止不主动关闭连接

[`Stop()`](internal/fileserver/tftp_server.go:104) 仅等待连接完成：

```go
func (s *TFTPServer) Stop() error {
    s.running = false
    s.connWg.Wait()
    return nil
}
```

**问题**: 如果有长时间传输，停止可能阻塞较久。

**建议**: 添加超时机制或主动关闭连接。
