## 2. 安全性审计

### 2.1 路径遍历防护 ✅ 优秀

#### TFTP路径安全

[`safePath()`](internal/fileserver/tftp_server.go:380) 函数实现完整路径遍历防护：

```go
func (s *TFTPServer) safePath(baseDir, filename string) (string, error) {
    // 清理路径
    filename = filepath.Clean(filename)
    
    // 移除开头的斜杠
    filename = strings.TrimPrefix(filename, string(os.PathSeparator))
    filename = strings.TrimPrefix(filename, "/")
    
    // 构建完整路径
    fullPath := filepath.Join(baseDir, filename)
    
    // 获取绝对路径
    absBase, err := filepath.Abs(baseDir)
    if err != nil {
        return "", fmt.Errorf("无法获取基础目录绝对路径")
    }
    
    absPath, err := filepath.Abs(fullPath)
    if err != nil {
        return "", fmt.Errorf("无法获取目标路径绝对路径")
    }
    
    // 检查目标路径是否在基础目录内
    if !strings.HasPrefix(absPath, absBase+string(os.PathSeparator)) && absPath != absBase {
        return "", fmt.Errorf("路径穿越攻击被阻止")
    }
    
    return absPath, nil
}
```

**评估**: 实现了完整的路径遍历防护，包括：
- 路径清理（`filepath.Clean`）
- 前缀剥离
- 绝对路径验证
- 前缀检查

### 2.4 安全风险点

#### ⚠️ 中风险：默认凭据硬编码

[`getDefaultConfig()`](internal/ui/fileserver_service.go:212) 中硬编码默认凭据：

```go
case "sftp":
    return &models.FileServerConfig{
        Username:    "admin",
        Password:    "admin",
        // ...
    }
```

**问题**: 默认凭据`admin/admin`可能被滥用。

**建议**: 首次启动时强制用户设置密码，或生成随机密码。

#### ⚠️ 低风险：HTTP默认无认证

HTTP服务器默认无认证：

```go
case "http":
    return &models.FileServerConfig{
        Username:    "",
        Password:    "",
        // ...
    }
```

**评估**: HTTP协议设计为轻量级文件浏览，无认证符合预期，但建议在文档中说明安全风险。

### 3.2 Panic恢复 ✅ 优秀

#### safeGo包装

[`safeGo()`](internal/fileserver/server.go:139) 实现panic恢复：

```go
func safeGo(name string, fn func()) {
    go func() {
        defer func() {
            if r := recover(); r != nil {
                logger.Error("FileServer", "-", "[%s] Goroutine 发生 panic: %v", name, r)
                logger.Error("FileServer", "-", "[%s] Panic 堆栈:\n%s", name, string(debug.Stack()))
                
                if manager := getGlobalManager(); manager != nil {
                    manager.emitLog(LogEvent{
                        Level:    LogLevelError,
                        Action:   ActionError,
                        Message:  fmt.Sprintf("[%s] 内部错误: %v", name, r),
                    })
                }
            }
        }()
        fn()
    }()
}
```

**评估**: 完整的panic恢复机制，并通知前端。

```go
func generateRandomPassword(length int) string {
    const charset = "abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789"
    b := make([]byte, length)
    rand.Read(b)
    for i := range b {
        b[i] = charset[b[i]%byte(len(charset))]
    }
    return string(b)
}
```

### 7.2 TFTP停止超时

```go
// 建议修改：添加停止超时
func (s *TFTPServer) Stop() error {
    s.mu.Lock()
    if !s.running {
        s.mu.Unlock()
        return nil
    }
    s.running = false
    s.mu.Unlock()
    
    // 添加超时等待
    done := make(chan struct{})
    go func() {
        s.connWg.Wait()
        close(done)
    }()
    
    select {
    case <-done:
        return nil
    case <-time.After(10 * time.Second):
        logger.Warn("FileServer:TFTP", "-", "停止超时，强制退出")
        return fmt.Errorf("停止超时")
    }
}
```
